# Personal Portfolio Webpage

> **🌟 Live Demo:** [View the Webpage Here!](https://SunnyTee2005.github.io/20260224-DIY/)

This is a personal portfolio webpage for Sunny Tee, created as part of an AIoT course assignment on February 24, 2026. 

## Features
- **Modern Design**: A dark-themed layout with vibrant accent colors.
- **Responsive Layout**: Adapts gracefully to both desktop and mobile screens.
- **Animations**: Smooth scroll-based fade-in effects and interactive hover states.
- **Custom Fonts**: Uses Google Fonts 'Outfit' for headings and 'Inter' for body text.

## Files Included
- `index.html`: The main HTML structure and content of the webpage.
- `styles.css`: The styling rules for the visual design and layout.
- `image1.jpg` & `image2.jpg`: Personal photos of Sunny Tee cycling.

## How to Run Locally
Simply open the `index.html` file in any modern web browser (like Chrome or Safari) to view the webpage directly on your computer. No server or complex setup is required.
