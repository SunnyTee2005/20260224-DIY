# Personal Portfolio Webpage

This is a personal portfolio webpage for Sunny Tee, created as part of a DIY exercise on February 24, 2026. 

## Features
- **Modern Design**: A dark-themed layout with vibrant accent colors.
- **Responsive Layout**: Adapts gracefully to both desktop and mobile screens.
- **Animations**: Smooth scroll-based fade-in effects and interactive hover states.
- **Custom Fonts**: Uses Google Fonts 'Outfit' for headings and 'Inter' for body text.

## Files Included
- `index.html`: The main structure and content of the webpage.
- `styles.css`: The styling rules for the design and layout.
- `summery.md`: This summary file outlining the project.
- Images: Personal photos of Sunny Tee cycling.

## How to Run
Simply open the `index.html` file in any modern web browser to view the webpage. No server or complex setup is required.
